# Ausardcompany Tap

## How do I install these formulae?

`brew install ausardcompany/tap/<formula>`

Or `brew tap ausardcompany/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ausardcompany/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
